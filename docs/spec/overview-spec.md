# Docker学習教材ウェブサイト - プロジェクト概要仕様

## 1. 背景・目的（Why）

### 背景
- Dockerはモダンな開発環境において必須のスキルとなっている
- 日本語で包括的にDockerを学べる教材が不足している
- 初心者から上級者まで一貫して学べるリソースが必要

### 目的
- Dockerの基礎から応用まで体系的に学習できるウェブサイトを提供する
- インタラクティブな学習体験により、効率的な知識習得を支援する
- 進捗管理機能により、継続的な学習をサポートする

## 2. 機能要件（What）

### 2.1 コンテンツ閲覧機能
- 全11チャプター、60以上のセクションの学習コンテンツを提供
- Markdown形式で記述されたコンテンツをHTML表示
- シンタックスハイライト付きのコードブロック表示
- 前後セクションへのナビゲーション

### 2.2 進捗管理機能
- セクション完了マーク機能
- チャプター別・全体の進捗率表示
- 最後に閲覧したセクションの記録
- LocalStorageによるデータ永続化

### 2.3 検索機能
- 全コンテンツを対象とした全文検索
- 検索結果のハイライト表示
- キーボードショートカット（Cmd/Ctrl + K）

### 2.4 ブックマーク機能
- 任意のセクションをブックマーク可能
- ブックマーク一覧ページ

### 2.5 UI/UX機能
- ダークモード/ライトモード切り替え
- レスポンシブデザイン（モバイル/タブレット/デスクトップ対応）
- サイドバーナビゲーション
- 目次（Table of Contents）表示

### 2.6 クイズ機能
- 各チャプター末尾に確認問題
- 選択式問題
- 即時フィードバックと解説表示

## 3. 非機能要件

- 初回読み込み時間: 3秒以内

- Lighthouse Performance スコア: 90以上

### 3.2 アクセシビリティ
- Lighthouse Accessibility スコア: 90以上
- キーボードナビゲーション対応
- スクリーンリーダー対応
- 適切なコントラスト比

### 3.3 ブラウザ対応
- Chrome（最新版）
- Firefox（最新版）
- Safari（最新版）
- Edge（最新版）

### 3.4 レスポンシブ対応
- モバイル: 320px〜767px
- タブレット: 768px〜1023px
- デスクトップ: 1024px以上
